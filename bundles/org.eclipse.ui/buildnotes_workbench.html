Eclipse SDK Build 134 - August 30, 2001

What's new in this drop

  • Font preference page added, including new header font.  Changed places which were hardcoding fonts to use these settings.
  • Changed Welcome page to use new header font.
  • Added printing to JFace TextViewer and to Eclipse UI AbstractTextEditor.
  • Fixes to some dialogs for potential sizing issues with NL strings.  See 1GH4KLL: ITPUI:WIN2000 - NLS test (JP) - Summary
  • Workbench-generated XML files now written and read using UTF8 encoding.
  • Support added for platform.ini (in addition to product.ini).

API additions

  • Added JFaceResources.HEADER_FONT and getHeaderFont().
